Step 1: Add the Script

Insert this script tag just before the closing </body> tag:

<script src="https://unpkg.com/@dappify/thirdweb@latest/dist/widgets.js"></script>

This enables all Dappify widgets on your page.

Connect Wallet Widget

Use this exact code to add the Connect Button (as seen in the Hero):

<div id="connect-button-hero"
     data-widget="ConnectButton"
     data-client-id="e9bae73377d43e42aea12c0d07474163"
     data-theme="dark"
     data-chains="43113,1,137"></div>

Claim NFT Widget

Use this exact code to add the Claim Button (as seen below):

<div id="claim-button-main"
     data-widget="ClaimButton"
     data-client-id="e9bae73377d43e42aea12c0d07474163"
     data-chains="43113"
     data-contract-address="0xeF155b056299Ba3e98987f37a2CB5518DE0a98F3"
     data-type="ERC721"
     data-claim-quantity="1"></div>
